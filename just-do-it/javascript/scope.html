# JS 作用域

JavaScript动态（解释执行）语言，不是提前编译的，编译结果也不能在分布式系统上移植

# 编译原理

  1. 分词/词法分析
  2. 解析/语法分析
  3. 代码生成

负责收据并维护所有声明的标识符（变量）的一系列查询，并有一定的规则来赋予这些标识符的访问权限

# LHS RHS

赋值操作的左侧或右侧

  • LHS(赋值操作的目标是谁)
  • RHS(谁是赋值操作的源头)

# 异常

  • 查询到作用域，引用值 null 或 undefined，TypeError
  • 未查询到作用域，ReferenceError

# 词法作用域

词法阶段的作用域

  • 全局环境作用域
  • 所执行的环境作用域

作用域查找会在招到第一个匹配的标识符停止（遮蔽效应） 作用域查找始终从运行时所处的最内部作用域开始，逐级向外或者向上进行，直到遇见第一个匹配的标识符为止
