# 伪元素 伪类

# 伪元素

伪元素是一个附加至选择器末的关键词，允许对被选择元素的特定部分修改样式

注意

一个选择器中只能使用一个伪元素，伪元素必须跟在普通选择器和基础选择器之后；使用双冒号（::）以便区分伪元素（::）伪类（:）

# 列举

  • ::after
  • ::before
  • ::backdrop
  • ::first-line
  • ::first-letter
  • ::cue
  • ::grammar-error
  • ::marker
  • ::placeholder --- input::placeholder
  • ::selection
  • ::slotted()
  • ::spelling-error
