# 快速排序算法

原地不稳定算法

  • 先分区，在处理子问题（由上到下）

# 快排实现

分区

  • 找基准值
  • 基准值小的放左边，基准值大的放右边
  • 递归两边（基准值）

# 时间复杂度

二叉树分析法

描述

n 个节点的二叉树，每个节点代表一个基准值，n 个节点代表了 n 个基准值

  • 选择基准值
    • 待排序区间尽量选择中间的数字，最好时间复杂度为 nlog2n
    • 待排序区间选择最大最小值，最差时间复杂度为 n^2

# 快排优化

  • 单边递归优化
  • 基准值取值优化，三点取中法
  • 分区操作优化

# 代码实现

  • 选择最左值作为基准值时，先从右边开始处理

  • 选择最右值作为基准值时，先从左边开始处理

  • 三点取中法（返回左右 依然遵守上方规则）
